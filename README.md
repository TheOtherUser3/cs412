# CS412 – Django Full-Stack Web Applications Portfolio

This repository serves as a portfolio of Django-based full-stack web applications developed over the course of CS412 at Boston University. It contains both guided class examples and homework projects, progressing from introductory Django concepts to a full-scale, personalized final project.

The primary project in this repository is the Snake AI Arena, located in the `/project` directory. Due to course requirements, all applications are maintained within this single repository.

---

## Capstone Project

### `/project` — Snake Arena (Final Project)
A full-stack web application where users can:
- Create AI-controlled snakes  
- Design custom game boards  
- Run automated snake battles  
- Evaluate performance to discover optimal strategies  

This project represents the culmination of the course and integrates:
- Backend simulation control  
- Game state management using the
  Django database to replay matches  
- Custom board generation  
- AI behavior logic  
- Web-based visualization

## Other Applications in This Repository

These projects were implemented as part of incremental learning throughout the course. Each explores a specific full-stack concept.

- **`hw`**  
  Introductory “Hello World” Django application used to learn project setup and request handling.

- **`quotes`**  
  *HW1*: Quote of the Day application focusing on routing, templates, and dynamic content rendering.

- **`formdata`**  
  Class example demonstrating form submission and POST handling

- **`restaurant`**  
  *HW2*: Simple restaurant ordering application implementing form logic and order processing.

- **`blog`**  
  Class example blog application focused on models, views, and relational data handling.

- **`mini_insta`**  
  *HW3+*: A mini Instagram-style application featuring:
  - User accounts  
  - Image posting  
  - Feed generation  
  - Following and liking
  <br>
  This is the most substantial project in the repository outside of the final project, with a focus on model interaction.

- **`marathon_analytics`**  
  Class example analytics application.

- **`voter_analytics`**  
  *HW8*: Voter analytics application that reads and processes CSV datasets to analyze voter distributions, with a search function.

- **`rest_framework`**  
  Class example introducing Django REST Framework concepts and API construction.

- **`dadjokes`**  
  *HW10*: A lightweight application that consumes and exposes external APIs to a very simple React Native application (found in the barebones "react" repository.)
